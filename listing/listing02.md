Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```
Основные положения работы с отложенными функциями:
* Выполнение происходит перед возвратом из функции;
* При вызове return, сначала вычисляется значение возвращаемых переменных, затем срабатывают отложенные функции (если не используются именованные возвращаемые параметры);
* Выполняются в очерёдности LIFO;
* Запоминают значения переменных, переданных в функцию, на момент объявления defer, а не в момент его вызова;
* При использовании замыкания defer использует значение, которое находится в переменной в момент вызова;
* Отложенные функции срабатывают даже при возникновении panic.

В функциях используется замыкание, значит в defer операции проводятся со значениями x в момент вызова отложенных функций.

В функции test() используется именованный возвращаемый параметр, значит defer может способен его изменить. Результатом функции будет 2.

Функция anotherTest() не использует именованных возвращаемых параметров. Возвращаемое значение предопределено и defer не может оказать на него влияния. Результатом функции будет 1.

Ответ:
```console
2
1
```
